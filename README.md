README
======

What is ThinOrm?
-----------------

ThinOrm is a tiny PDO wrapper and helper to avoid huge ORM's inefficiency.

Requirements
------------

ThinOrm is only supported on PHP 5.3.3 and up.

Installation
------------

The best way to install ThinOrm using Composer.
